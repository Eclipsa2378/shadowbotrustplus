# Документация команд

> Команды могут выполняться через Discord или во внутриигровом командном чате. Чтобы запускать Slash-команды в Discord, вы должны иметь назначенную роль Discord для бота. Если роль для бота не задана, то все могут использовать Slash-команды по умолчанию. Чтобы запускать внутриигровые команды, вы должны быть в одной внутриигровой команде с хостером. Внутриигровые команды можно запускать только из командного чата, а не из глобального. Вы также можете запускать внутриигровые команды из текстового канала Discord `commands`.

- [Discord Slash команды](commands.md#discord-slash-commands)
- [Внутриигровые команды](commands.md#in-game-commands)

## **base**

> **Система баз (9 квадратов).** База определяется как центральный квадрат плюс 8 квадратов вокруг. Бот отправляет уведомления о входе/выходе (1 раз в 15 сек). При смене сервера базы удаляются.
<br>Команда: `!base create <название>`
<br>Команда: `!base delete <название>`
<br>Команда: `!base list`
<br>Команда: `!base players`

Подкоманда | Описание | Обязательно
---------- | ----------- | --------
`create` | Создать базу (`!base create <название>`). | `True`
`delete` | Удалить базу (`!base delete <название>`). | `True`
`list` | Показать список баз (`!base list`). | `False`
`players` | Показать игроков на базах (`!base players`). | `False`



## **clan**

> **Клановые уведомления.** Бот уведомляет о различных событиях команды: все игроки на одной базе, команда разделилась по монументам, игроки слишком далеко друг от друга, все игроки онлайн, один игрок остался на монументе.
<br>Команда: `!clan toggle`

Подкоманда | Описание | Обязательно
---------- | ----------- | --------
`toggle` | Переключить уведомления (`!clan toggle`). | `False`
