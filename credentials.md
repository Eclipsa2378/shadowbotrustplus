# Документация по учетным данным

> Для корректной работы экземпляра бота необходимо предварительно настроить учетные данные. Их добавление позволяет устанавливать соединение с серверами и получать доступ к связанным функциям.

## Получение учетных данных

1. Загрузите и установите [программу подключения Rust+](https://shadowbot.eclipsa.ru/).
2. После завершения установки нажмите кнопку **«Connect with Rust+»**.
3. В открывшемся окне выполните вход в свою учетную запись **Steam**.
4. После успешной авторизации отобразится окно со слэш-командой вида
   `/credentials add ...`.
5. Нажмите кнопку **«Копировать»**, чтобы скопировать команду.
6. Вставьте скопированную слэш-команду в любой текстовый канал вашего Discord-сервера, к которому у бота есть доступ. Выбор конкретного канала не имеет значения.

Данный процесс является одинаковым как для владельца бота, так и для участников команды, регистрирующих собственные учетные данные (в основном для расширения возможностей использования команды `leader`).

После регистрации учетных данных участнику команды необходимо войти в игру и подключиться к серверу. Информацию о том, какие участники команды подключены к серверу, можно увидеть в информационном канале: рядом с их именами должна отображаться иконка **P** (см. ниже).

## Назначение учетных данных

Учетные данные необходимы для получения следующих уведомлений:

* уведомления о подключении к серверу;
* уведомления о подключении к умным устройствам;
* уведомления от умных сигнальных тревог;
* уведомления о смерти игрока в автономном режиме;
* уведомления о входе в систему участников команды;
* новости от Facepunch.

Без настроенных учетных данных бот не сможет функционировать корректно.
