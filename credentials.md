# Документация по учетным данным

> Чтобы правильно использовать экземпляр бота, необходимо настроить учетные данные. Добавление этих учетных данных позволяет вам соединяться с серверами, а также подключаться к ним.

## Получение учетных данных

* Загрузите и установите [Сайт для работы с учетными данными rustplusplus](https://shadowbot.eclipsa.ru/)
* После загрузки и установки откройте его и нажмите кнопку «Connect with Rust+» (Подключиться с Rust+).

* Откроется другое окно, в котором вам нужно будет войти в свою учетную запись Steam.

* После входа в систему вы увидите окно со слэш-командой /credentials add .....
* Нажмите «Копировать», чтобы скопировать слэш-команду.

* Вставьте слэш-команду в текстовый канал вашего Discord-сервера. Неважно, в каком текстовом канале вы запускаете слэш-команду, главное, чтобы бот имел к нему доступ. 

* Этот процесс одинаков как для владельца бота, так и для любого члена команды, который хочет зарегистрировать свои учетные данные (в основном с целью расширения использования команды leader). После того, как член команды зарегистрировал свои учетные данные, ему нужно просто зайти в игру и соединиться с сервером. ## Зачем нужны учетные данные?

Учетные данные необходимы для получения следующего:

* Уведомления о соединении с сервером
* Уведомления о соединении с умными устройствами
* Уведомления об умных сигналах тревоги
* Уведомления о смерти игрока в автономном режиме
* Уведомления о входе в систему товарища по команде
* Новости Facepunch

Без них бот не будет работать должным образом.
