# Подключение к серверу

1. Убедитесь, что учетные данные настроены корректно.

2. Войдите на сервер **Rust**, к которому вы планируете подключиться.

3. Нажмите клавишу `ESC`, затем выберите пункт **Rust+**.

4. Нажмите кнопку **«Подключиться к серверу»**.

5. После успешного подключения информация о сервере отобразится в текстовом канале Discord `servers`.

6. В этом канале нажмите кнопку **«Подключиться»**, после чего бот автоматически подключится к выбранному серверу Rust.
